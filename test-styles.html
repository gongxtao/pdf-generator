动态样式系统测试文档

这是一个测试文档，用于验证Tiptap编辑器的动态样式提取和应用功能。该文档包含了多种复杂的CSS样式，包括CSS变量、渐变背景、自定义字体、伪元素等。

主要特性

  • CSS变量支持
  • Google字体集成
  • 渐变背景和阴影效果
  • 响应式表格设计
  • 交互式按钮样式
  • 自定义列表样式

样式变量系统

本文档使用了CSS变量来管理颜色主题，包括主色调 --primary-color、次要色调 --secondary-color 和强调色 --accent-color。

动态样式系统能够100%还原HTML文档的原始样式，无论多么复杂的CSS规则都能正确解析和应用。

数据表格示例

作用域限定 完成 将样式限定在编辑器范围内

:root { --primary-color: #2563eb; --secondary-color: #f59e0b; --accent-color: #10b981; } .highlight { background: linear-gradient(120deg, #a8edea 0%, #fed6e3 100%); padding: 20px; border-radius: 8px; }
